# 导入第三方的网络请求的包

首先在项目根目录初始化一个package.json的包，并安装

npm init -y
# 安装
npm install @escook/request-miniprogram

在项目main.js中

// 导入网络请求的包
import { $http } from '@escook/request-miniprogram'

// 在 uni-app 项目中，可以把 $http 挂载到 uni 顶级对象之上，方便全局调用
uni.$http = $http

// 请求拦截器
$http.beforeRequest = function() {
  uni.showLoading({
    title: '数据加载中...'
  })
}

// 响应拦截器
$http.afterRequest = function () {
  uni.hideLoading()
}

在 JavaScript 中，使用 async 和 await 关键字可以方便地处理异步操作。这些关键字通常用于异步函数中，下面是它们的作用：

  1. async 关键字：async 用于声明一个函数是一个异步函数，该函数返回一个 Promise 对象。在异步函数内部，你可以使用 await 关键字来等待异步操作完成。
  2. await 关键字：await 用于等待一个异步操作的完成，它只能在异步函数内部使用。当遇到 await 表达式时，代码的执行会暂停，直到该表达式的 Promise 对象状态变为 resolved（已解决）或 rejected（已拒绝）。同时，await 表达式会返回被解析的值，并且可以将其赋值给一个变量。

通过使用 async 和 await，你可以编写更加直观和易读的异步代码，而无需使用回调函数或者 Promise 的链式调用。它们提供了一种类似于同步代码的写法，使得异步操作的处理更加简洁和可读性更高。

需要注意的是，await 只能在异步函数内部使用，因此如果你想使用 await，你需要将方法声明为 async 函数。
